# C++로 구현한 간단한 Galaga

비트 플래그의 활용법과 비트 연산의 개념을 이해하기 위해, 콘솔 앱으로 간단하게 제작한 C++ Galaga입니다.\
프로젝트를 진행하며 비트플래그 뿐 아니라 상태 관리, 라이프 사이클의 중요성, 객체를 사용하는 이유 등에 대해 학습할 수 있었습니다.

### 학습 목표
* 비트 연산의 원리 이해
* 비트플래그의 실질적인 활용 방법 이해

### 구현 목표
* 비트 플래그를 이용한 오브젝트의 상태 관리
* 실제 입력을 기반으로 상호작용 가능한 Galaga 스타일의 미니게임

### 추가 내용 및 배운 점
코드리뷰를 통해 struct를 사용하여 오브젝트를 관리하고, 코드의 가독성을 높이는 것이 좋겠다는 조언을 받았습니다.\
이를 토대로 기존에 펼쳐진 배열로 관리되던 오브젝트들을 struct로 묶어 조금 더 이해하기 편하도록 코드를 수정했습니다.\
이 과정에서 기존에 사용하던 비트플래그는 더 이상 사용할 필요가 없었습니다.\
그러나 코드를 리팩토링하는 과정에서 비트플래그는 객체가 여러 상태에 대한 정보를 가질 때 유효하게 사용할 수 있다는 점을 느꼈습니다.\
라이프 사이클 설계의 중요성과 가독성이 높은 코드의 필요성 등을 체감했습니다.\
비트플래그 학습을 위해 시작한 프로젝트였지만, 코드리뷰를 통해 객체를 사용하는 이유와 리팩토링 과정을 경험할 수 있었습니다.
